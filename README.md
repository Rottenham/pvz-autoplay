# 挂机脚本库

## 依赖环境

```
AvZ2 240510
```

## 使用方法

除脚本源码外，各文件夹内也提供了阵型存档，阵图，以及我进行的万 f 测试结果。

运行脚本前，需先将 `inc/custom` 置于 `AsmVsZombies2/inc` 目录下。

部分脚本在运行万 f 测试后进行了微改。由于显然的原因，笔者不可能再跑一遍万 f 测试（<del>跑一遍很累的你造吗！</del>）。若遇脚本 bug，欢迎联系本人反馈。
